Story

When we want news about technologies, we used to go to different websites, but we rarely find useful content and very little on social media. We used to take a lot of time to get useful information from these and also have to go to a lot of blogs which eats our time bit by bit. Then we recognized the core problem.

Problem

Socials contain information that contains not only useful information but also contains useless information like un-updated content, fake posts, etc.

solution

We are designing a platform that helps technology enthusiasts, developers, entrepreneurs and students to get updated content, news as well as messages which only allow technology-related content which is processed by Machine intelligence.

features

  • Communities that help users to get genuine content.
  • Powerful messaging across users, groups and communities.
  • Posts and Story creation.
  • Technology based filtering of your prefferd choice.
  • Exploring features that helps to get top useful content like Trending, popular, announcements, startup events, product updates, etc.
